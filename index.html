FrozenJS 是针对移动端开发的 js 组件库，其依赖 zepto.js 和 FrozenUI。

FrozenJS 包括： basic：FrozenJS 的一些基础功能，包括模板引擎、tap 支持等。
ui：主要是一些触屏常用的 UI 组件，包括 dialog 等。

effect：非常用的特效库，特殊场景使用到是可以单独调用。

兼容android 2.3 +，ios 4.0 + 。
